# TytlLab Marketing Site (v4)

Static site, no build step. Open `index.html` in a browser, or serve the folder with any static server.

## Pages

- `index.html` — home (hero with Desktop/Mobile dashboard preview, skyline background, assistant card deck with unstack/restack, pricing, CTA)
- `about.html`, `blog.html`, `contact.html` — interior pages, same design system

## Assets

- `logotransparent.png` — TL logo
- `dashss.png` — desktop dashboard screenshot (hero preview)
- `TytlLab_Mobile_Dashboard_Mock.jpg` — mobile dashboard (hero preview)
- `phase1tilo.mp4`, `phase2tilo.mp4`, `phase3tilo.mp4` — hover-play videos on assistant cards 1–3

## Design system

- Headings: Instrument Serif (400), italic + blue accents. Body: Inter. (Google Fonts, loaded via link tags.)
- Colors: background #F6F8FA, primary blue #1067FE, orange accent #FF8400, black buttons #000, borders #E7EAF0.
- Cards: white, 24px radius, soft shadows. Pastel gradient blurs behind hero sections.
